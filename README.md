# HyperOS Old Control Center

[English](#english) | [Русский](#русский)

---

## English

**HyperOS Old Control Center** is a Magisk/KernelSU/APatch module that forcefully restores the legacy Control Center style toggle in HyperOS settings using a static overlay.

### Description
In some HyperOS builds, Xiaomi removed the option to switch back to the classic Control Center layout. This module bypasses that restriction by injecting a static overlay into the system framework, bringing back the "Control center style" menu.

### Features
- **Toggle Restore:** Re-enables the "Control center style" selection in Settings.
- **Static Overlay:** Uses a stable method to override system resources.
- **Systemless:** Does not modify `/system` files directly.

### Installation
1. Download the latest module `zip`.
2. Flash it via Magisk, KernelSU, or APatch.
3. Reboot.
4. Navigate to **Settings -> Notifications & Control center -> Control center style**.

## Русский

**HyperOS Old Control Center** — это модуль для Magisk/KernelSU/APatch, который принудительно возвращает выбор старого стиля Центра Управления в настройках HyperOS через статический оверлей.

### Описание
В определенных билдах HyperOS возможность переключиться на старый вид ЦУ была вырезана. Этот модуль исправляет эту несправедливость, подкидывая системе статический оверлей, который разблокирует соответствующий пункт в настройках.

### Фишки
- **Возврат выбора:** Оживляет меню «Стиль Центра управления».
- **Static Overlay:** Самый надежный метод подмены системных ресурсов.
- **Systemless:** Никакой грязи в системном разделе, все чисто.

### Установка
1. Качаешь архив с модулем.
2. Прошиваешь через Magisk, KernelSU или APatch.
3. Ребутишь девайс.
4. Идешь в **Настройки -> Уведомления и Центр управления -> Стиль Центра управления**.
